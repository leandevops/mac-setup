![macOS](https://github.com/lestex/.dotfiles/.github/workflows/mac.yaml/badge.svg)

## Installation
Run:
```sh
➜ bash <(curl -s https://raw.githubusercontent.com/lestex/dotfiles/master/install.sh)
➜ cd .dotfiles && make
```
